A full-stack web application designed to streamline **transport logistics and consignment management** for rural youth initiatives.  
**Deployed on Vercel: (https://transport-computerized-software-kg7.vercel.app/)**
Backend  url:https://transport-computerized-software.onrender.com

## ‚ú® Features

- **Role-based dashboards**  
  - Customer: Create consignments with minimal input (volume & destination).  
  - Manager: Assign consignments to drivers, track statuses.  
  - Driver: View assigned consignments and update delivery progress.  

- **Consignment workflow automation**  
  - Simple, frictionless submission process.  
  - Real-time updates across dashboards.  

- **Backend/Frontend integration**  
  - RESTful APIs with schema validation.  
  - Secure token-based authentication.  

- **Responsive UI**  
  - Built with React for a clean, user-friendly experience.  
  - Optimized for desktop and mobile.
